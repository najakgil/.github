## 📌 Introduce
### ✅ 서비스 소개
> **나작길 | 나만의 작은 총장님 만들기**

✌️**이길여 총장님**을 모델로,   
다양한 스타일과 개성으로 손쉽게 **나만의 총장님을 꾸며 볼 수 있는 웹 서비스**입니다.

### ✅ 기획 의도
'학교 생활에 접목하는 IT'라는 주제로, 아래와 같은 이유로 서비스를 기획하였습니다.

**1. 가천대생이라면 모두가 관심있어 하는 이길여 총장님**    
'대학 축제에서 싸이춤 추는 총장님'이라는 제목의 유튜브 영상이 **343만회 조회수**(23.11.14 기준)를 기록함.   
해당 영상이 게시되어 있는 유튜브 가천대학교 계정의 최근 동영상 12개의 평균 조회수는 5.7천 회라는 점으로 고려하였을 때, 높은 조회수임을 확인할 수 있음.

**2. 누구보다도 주도적인 Z세대**    
Z세대의 경우, **자신이 만든 콘텐츠를 널리 확산시키고자 하는 트렌드**가 있음.   
예를 들어 '아디아디 챌린지'에 참여한 콘텐츠는 2.7만여 개로 '아며들었다'며 각종 숏폼 플랫폼을 달군 적이 있음.
